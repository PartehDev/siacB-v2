Docs for siacB v2, syntax for beginning and end inspired by ArnoldC

Getting Started

To get started, download the latest stable build of siacB. This programming language is kinda like C, mixed with beginning keywords like ArnoldC. When siacB is fully downloaded, open your text editor and make a hello world file with the following content:

WAKEY WAKEY siacb::test BEDTIME
Congrats! You have made a siacB program! Now, save it as whatever you like! e.g test.sb, etc. MAKE SURE IT ENDS WITH .SB!













Running Code

To run your code, simply run main.py, and type , without the .sb part. Your file should run!
